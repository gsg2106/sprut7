Ввод показаний и получение счета.

Для проведения расчетов за потребленную энергию через WEB  услугу вам выделяется определенный временной период. Во время данного расчетного периода Вы можете войти в пункт меню “Ввод показаний” и выполнить расчеты. В остальное время вход в  пункт меню “Ввод показаний” закрыт.

Формирование бланка рапорта.

При первом входе в пункт меню “Ввод показаний”  вы видите показания по рапорту предыдущего отчетного периода. Данные показания Вам могут пригодиться для контроля. Чтобы  ввести показания текущего отчетного периода нажмите кнопку “Сформировать рапорт за ……”. По окончании расчета Вы получите на экране бланк рапорта за текущий отчетный перид для последующего ввода показаний. Кнопка  “Сформировать рапорт за ……” поменяет название на “Переформировать рапорт за ….”. В дальнейшем эту кнопку нажимать не следует до наступления следующего отчетного периода.

Ввод показаний.

Ввод показаний выполняется только в те строки рапорта, которые имеют состояние “Введите показания”. В прочие строки показания не вводятся по ряду причин (например: отключенные точки учета или расчет по установленной мощности). Чтобы ввести показания выполните двойной клик по соответствующей строки или после выбора требуемой строки простым кликом нажмите кнопку “Исправить”. Результат будет одинаковым. Откроется диалог ввода показаний в строку.

Формирование счета на оплату.

По окончании ввода показаний, нажмите кнопку “Получить счет”. По окончании процесса формирования документов, появится дополнительная закладка “Счета на оплату” с опубликованным для скачивания  списком сформированных документов.  Скачать документы можно либо с помощью ссылки на файл либо кликом по кнопке “Скачать”.  В последнем случае открывается диалог скачивания файла (Учтите, что данная функция может быть блокирована браузером, если установлен соответствующий запрет).  Скачайте все опубликованные файлы для их последующей подписи.

Подписать документ.

Нажмите кнопку “Подписать счет”.  Откроется дополнительная закладка “Подписать файл счета”.  Здесь вам следует выбрать один из скачанных файлов и подписать его. Собственной электронной подписью. Попутно вы можете посмотреть содержание файла и проверить наличие электронной подписи со стороны энергоснабжающей организации. Затем вам следует отослать подписанный файл назад на сервер энергоснабжающей организации. Данную процедуру необходимо повторить со всеми скачанными файлами. Учтите, что название подписанного файла отличается от исходного. В название добавлен суффикс _sign.pdf.

Отправка файла в энергоснабжающую организацию.

Нажмите кнопку “Отправить файл в э/c орг.”. Откроется диалог выбора файла. Выберите один из подписанный файлов и дождитесь окончания операции. Если вы все сделали правильно, то вы получите сообщение об успешно проведенной операции. И в списке опубликованных файлов появится птичка о наличии подписанного файла.

Расчет считается завершенным, когда все опубликованные файлы помечены соответствующими птичками. На этом процесс  расчета завершен.
